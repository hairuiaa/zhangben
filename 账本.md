# 账本

> 用 Markdown 表格记账。每次有新消费，直接在「流水明细」里新增一行即可。

## 使用规则

- 金额统一用人民币或泰铢时，请在「币种」列写清楚，例如：`CNY` / `THB`。
- 支出金额写负数，例如 `-35`；收入金额写正数，例如 `5000`。
- 「账户」可以写：现金、支付宝、微信、银行卡、信用卡等。
- 「分类」可以写：餐饮、交通、住宿、购物、娱乐、门票、药品、杂项等。
- 每月底可以手动汇总到「月度汇总」。
